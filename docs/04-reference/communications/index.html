Communications Reference

Staying in contact in remote Australia. From the standard UHF radio in every 4WD to satellite communications and emergency beacons.

Primary Content

  • UHF CB Radio Guide — Complete 80-channel reference. Emergency channels, convoy protocols, antenna setup, range. No licence required. Standard in all Australian 4WDs.
  • 2M Amateur Radio Repeaters — 146 repeaters across 8 states. Requires Amateur Operator's Certificate.

Pages

  • Australian 2m Amateur Radio Repeaters

    Comprehensive directory of Australian VHF 2-metre (144–148 MHz) amateur radio repeaters, organised by state and territory. Includes band plans, CTCSS tones, IRLP/Echolink nodes, WICEN emergency frequencies, APRS digipeaters, and practical programming notes.

  • UHF CB Radio Guide

    Complete Australian UHF CB radio reference. All 80 channels, emergency channels, convoy protocols, antenna setup, range, and practical outback use. No licence required.
