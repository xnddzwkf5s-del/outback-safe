# Outback Safe — Australian Remote Travel Reference

Your offline survival reference for remote Australian 4WD travel. Works on iPhone, iPad, desktop, and USB stick. No internet required.

## What's Inside

- **🚙 Outback 4WD Survival** — vehicle breakdown, recovery, bush mechanics, heat survival, water crossings, sand driving, croc safety, fuel management, route planning, grab bag
- **🏥 Medicine & First Aid** — snake bite, spider bites, marine stingers, tick bites, anaphylaxis, bush medicine, remote first aid techniques
- **🗺️ Maps & Track Guides** — Simpson Desert, Cape York, Canning Stock Route, Gibb River Road, Anne Beadell Highway, Binns Track + 9 other tracks
- **📻 Communications** — UHF CB (80 channels, no licence), 2M amateur repeaters
- **🔥 Survival Basics** — water, fire, shelter, food, signaling, navigation, group management

## How to Build

```bash
python3 build.py
```

Outputs to `site/` — open `site/index.html` in any browser.

## How to Use

1. Download the latest `Outback Safe.zip` from Releases
2. Unzip and open `index.html` — works offline immediately
3. On iPhone: AirDrop the ZIP, unzip in Files app, tap to browse
4. On USB: copy the `site/` folder, works on any computer with a browser
