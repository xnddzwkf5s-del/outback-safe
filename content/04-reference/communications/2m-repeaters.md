> **⚠️ Amateur Licence Required:** Operating on 2M (144–148 MHz) amateur radio frequencies requires an Amateur Operator's Certificate (Foundation, Standard, or Advanced). Using these frequencies without a licence is illegal in Australia and carries significant penalties. For licence-free communication, see the **[UHF CB Radio Guide](uhf-cb-guide)** — the standard communication system in all Australian 4WDs.

# Australian 2m (VHF 144–148 MHz) Amateur Radio Repeaters

A comprehensive reference of amateur radio repeaters across Australia, organised by state and territory. This guide covers the 2-metre band (144–148 MHz), the most popular VHF band for local and regional communication.

All frequencies are in MHz. Standard Australian 2m repeater offset is **–600 kHz** (output minus 600 kHz = input frequency) unless otherwise noted.

---

## 2m Band Plan Summary (VK Standard)

The Australian 2m band plan divides the spectrum for different modes:

| Range (MHz) | Primary Use |
|---|---|
| 144.000 – 144.100 | CW (Morse code) only |
| 144.100 – 144.400 | CW and SSB weak-signal work |
| 144.400 – 144.800 | Beacons (automated propagation beacons — no voice) |
| 144.800 – 145.000 | Digital modes (APRS on 145.175 MHz, packet radio) |
| 145.000 – 146.000 | **Repeater inputs** (listening on repeater output, transmitting here) |
| 146.000 – 147.000 | **Repeater outputs** (what you listen to for repeaters) |
| 147.000 – 148.000 | Simplex FM, some repeaters, experimental |

---

## APRS Frequency

- **145.175 MHz** — Australian APRS (Automatic Packet Reporting System) national frequency
- Recommended APRS digipeater path for outback travel: `WIDE1-1,WIDE2-2`
- APRS in Australia uses 1200 baud AFSK (Bell 202 standard)
- Mobile APRS: consider a dedicated 2m antenna for reliable tracking

---

## Emergency & Calling Frequencies

| Frequency | Purpose |
|---|---|
| **146.500 MHz** | National 2m FM calling frequency — monitor this when mobile |
| **145.500 MHz** | Alternate portable/mobile FM calling frequency |
| **439.000 MHz** | National 70 cm FM calling frequency |
| **146.750 MHz** | Often used as a roadside/travel assistance frequency |
| **147.000 MHz** | National VHF simplex emergency frequency (suggested) |

**Emergency procedure:** Tune to the calling frequency, listen for a moment, then call "MAYDAY" or "PAN PAN" followed by your situation and location. In a genuine life-threatening emergency, **any person may use amateur frequencies** regardless of licence status (per ACMA regulations).

## Practical Notes

### Licensing

To transmit on amateur radio frequencies, you need an Australian amateur radio licence issued by the ACMA:

| Licence Class | Max Power | Equipment | Bands |
|---|---|---|---|
| Foundation | 10 W | Type-approved commercial equipment only | Limited HF + all VHF/UHF |
| Standard | 100 W (25 W for Foundation-only bands) | Home-built allowed with inspection | All amateur bands |
| Advanced | 400 W | Full privileges | All amateur bands |

Foundation licence is the entry point — a one-day course and exam. No Morse code requirement in Australia.

### Programming Radios for Australian Repeaters

**Standard 2m repeater setup (for most Australian repeaters):**

1. Set frequency mode to VFO
2. Enter the repeater **output frequency** (what you listen to)
3. Set offset direction to **–** (negative/minus)
4. Set offset to **600 kHz**
5. Enable CTCSS (if required) and set tone to **91.5 Hz**
6. Save to a memory channel
7. Repeat for each repeater along your planned route

**Example: Programming VK2RMP (Maddens Plains, 146.850 MHz)**

- RX frequency: **146.850 MHz**
- TX frequency: **146.250 MHz** (146.850 – 600 kHz)
- CTCSS: **91.5 Hz**
- Tone mode: **Tone** (not TSQL/CTCSS decode unless required)
- Name: `VK2RMP`

### Common CTCSS Tones for Australian Repeaters

| Tone (Hz) | Use |
|---|---|
| **91.5** | Most common VK standard; used by the majority of repeaters |
| **123.0** | Second most common; some regional repeaters |
| **131.8** | Less common; some VK4 and VK6 sites |
| **141.3** | Some older commercial-install repeaters |
| **None (CSQ)** | Open carrier squelch — no tone required; listen first to check |

Many Australian repeaters are "open" (no tone required). Always try without CTCSS first. If you hear a repeater but can't access it, the tone is likely required for transmit.

### Radio Etiquette

- **Monitor before transmitting** — listen for several seconds to ensure the frequency is clear
- **Identify** — give your callsign at the start and end of each transmission (every 10 minutes during a long QSO)
- **No "kerchunking"** — don't key up without identifying; it's poor etiquette and technically illegal
- **Pause between transmissions** — allow the repeater timeout to reset
- **Emergency traffic has priority** — if you hear "MAYDAY" or "PAN PAN", clear the frequency immediately
- **Simplex first** — if you can communicate without a repeater, do so; leave repeaters available for those who need the range

### Emergency Use of Amateur Radio

In a genuine life-threatening emergency, **any person may use amateur radio frequencies** regardless of licence status. The ACMA recognises this under the Radiocommunications Act. If you're not licensed but need to call for help:

1. Set the radio to **146.500 MHz** (national calling frequency)
2. Key the microphone and say:
   - **"MAYDAY, MAYDAY, MAYDAY"** (life-threatening)
   - **"PAN PAN, PAN PAN, PAN PAN"** (urgent but not life-threatening)
3. State your location
4. Describe the emergency
5. Release the PTT and listen for a response
6. If no response on 146.500, try **146.750 MHz** or any known local repeater output frequency

### Antenna Notes

- **Quarter-wave whip:** simplest; ~50 cm for 2m; good omnidirectional coverage; ideal for mobile use
- **Half-wave vertical:** slightly more gain; ~1 m for 2m; requires a ground plane (roof mount works)
- **5/8-wave vertical:** higher gain (~3 dBd), ~85 cm for 2m; excellent for mobile; slightly narrower radiation angle
- **J-pole:** popular for base stations; no ground plane required; ~1.5 m total
- **Yagi (beam):** directional; 3–10 elements for 2m; 2–8 m boom length; used for DX or weak-signal work
- **Magnetic mount:** adequate for temporary use; reduced efficiency vs. through-roof mounting
- **NMO mount:** professional standard for permanent vehicle installation
